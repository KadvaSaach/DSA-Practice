## Time complexity

### Average

| Access | Search | Insertion | Deletion |
| :----: | :----: | :-------: | :------: |
|  O(1)  |  O(n)  |   O(n)    |   O(n)   |

### Worst

| Access | Search | Insertion | Deletion |
| :----: | :----: | :-------: | :------: |
|  O(1)  |  O(n)  |   O(n)    |   O(n)   |

## Space complexity

O(n)

### Advantages vs Disadvantages

| Advantages     | Disadvantages |
| -------------- | ------------- |
| Fast Lookups   | Slow insert   |
| Fast Push/Pops | Slow delete   |
| Ordered        | Fixed Size*  |

*if using static array
